변수 선언 방법
	-자료형 변수명;

자료형
	-기본형
		-값을 저장
		-문자, 정수, 실수, 논리
	-참조형
		-주소를 저장
		-기본형 이외


char
	-2byte 	(한글 표현 가능)
int	
	-4byte
double
	-8byte	(float보다 정확)
boolean
	-1byte   	true, false 저장

변수명 작성 규칙
	-대소문자 구별, 특수문자(_$만), 키워드 사용불가, 중복선언 불가, 첫글자 숫자x
변수명 작성 관례
	-이름을 의미있게, 카멜 표기법에 맞춰(소문자로 시작, 두번째 단어부터 대문자로), 클래스 이름은 대문자로 시작, 상수는 대문자

자료형변환
	-다른 타입의 자료를 저장 / 활용 할 때

	-자동
		-작은 자료를 큰 자료에 저장
	-강제(명시적)
		-큰 자료를 작은 자료에 저장
